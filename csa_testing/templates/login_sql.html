Как провести атаку:

  1. Введите любой существующий логин из таблицы справа
  2. В поле пароля введите: ' OR '1'='1
  3. Нажмите "Войти" - вы получите доступ как первый пользователь в БД

Почему это работает?

Сервер формирует SQL-запрос: SELECT * FROM auth_user WHERE username = '[логин]' AND password = '[пароль]'

С вашим вводом запрос становится: SELECT * FROM auth_user WHERE username = 'admin' AND password = '' OR '1'='1'

Условие '1'='1' всегда истинно, поэтому запрос возвращает всех пользователей!
